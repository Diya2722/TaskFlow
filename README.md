# TaskFlow 🚀

**A modern, beautiful, and full-featured Task Management Application** with Kanban Board, Calendar, Dashboard, and more.

![TaskFlow Banner](https://via.placeholder.com/1200x400/7C3AED/FFFFFF?text=TaskFlow+-+Stay+Organized)

## 📋 About The Project

**TaskFlow** is a complete full-stack task management web application designed to help individuals and teams stay organized and productive. It combines the power of a **drag-and-drop Kanban board**, a **visual Calendar**, and a rich **Dashboard** with statistics — all in a sleek, responsive, and dark-mode-enabled interface.

Whether you're managing personal tasks or working in a team, TaskFlow provides everything you need to track, prioritize, and complete your work efficiently.

---

## ✨ Key Features

### Core Functionality
- **Kanban Board** – Drag & drop tasks between columns (To Do, In Progress, Done)
- **Calendar View** – See due dates visually with task indicators
- **Dashboard** – Overview with stats, filters, search, and sorting
- **Task Management** – Create, edit, delete, and mark tasks as complete
- **Comments System** – Add comments on tasks for better collaboration
- **Labels & Priorities** – Organize tasks with custom labels and priority levels (High, Medium, Low)

### Additional Features
- **Advanced Filters** – Filter by status, priority, due date (Overdue, Today, This Week)
- **Real-time Search** – Search tasks by title or description
- **Dark Mode** – Beautiful light & dark theme support
- **Responsive Design** – Works perfectly on desktop, tablet, and mobile
- **Keyboard Shortcuts** – Press `N` to create new task quickly
- **Export Tasks** – Download all tasks as JSON
- **Secure Authentication** – Register/Login with "Remember Me" option

## 🛠 Tech Stack

### Frontend
- **React 18** + **Vite**
- **Tailwind CSS**
- **React Router DOM**
- **Lucide React** (icons)
- **React Calendar**
- **Axios** + **React Hot Toast**

### Backend
- **Node.js** + **Express**
- **MongoDB** + **Mongoose**
- **JWT Authentication**
- **bcryptjs** (password hashing)
- **Zod** (validation)
- **Cookie-based auth**
